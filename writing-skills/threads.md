# Threads Writing Pattern

Threads is conversational public thinking. It should feel like a clear thought people can reply to, not a formal announcement.

## Adaptation

- Make the persona more conversational and present.
- Keep one idea per post.
- Turn long source content into a short thread only when sequence helps.
- Use minimal or no hashtags.
- Questions should feel real, not forced.

## Length

- Single post: 120-350 characters when possible
- Thread: 2-5 posts
- One idea per post

## Patterns

Single thought:

```text
[Observation, opinion, or tension]
[Specific detail or implication]
[Optional reply-friendly question]
```

Mini-thread:

```text
1. [Standalone main thought]
2. [Why it matters]
3. [Example or lesson]
4. [Soft CTA or question]
```

Blog-to-Threads:

```text
1. [Most discussable takeaway]
2. [Useful detail from the blog]
3. [Human POV from persona]
4. [Link or full-breakdown note]
```

## Check

- First post stands alone.
- Each post adds one new idea.
- Persona sounds human, not corporate.
- Campaign intent remains recognizable.
