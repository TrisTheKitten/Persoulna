# Telegram Writing Pattern

Telegram is direct channel communication. It should feel like a useful message to an audience that already chose to hear from the user.

## Adaptation

- Make the persona more direct.
- Keep the template's order only if it improves clarity.
- Put the link or next step where it is easy to tap.
- Use bullets when scanning matters.
- Avoid social-media performance language and hashtag blocks.

## Length

- Alert/update: 30-90 words
- Community announcement: 80-180 words
- Digest: 150-400 words

## Patterns

Announcement:

```text
[Direct headline]
[What changed / what is available]
[Why it matters]
[Link or next step]
```

Digest:

```text
[Digest title]
[3-5 useful bullets]
[Best next action]
```

Image with message:

```text
Caption:
[Short context or result]

Follow-up message:
[Full explanation]
[Action link]
```

## Check

- Opening tells readers why this matters.
- Message is useful without attention tricks.
- Link or action is obvious.
- Persona sounds direct but still recognizable.
